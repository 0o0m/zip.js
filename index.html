zip.js by gildas-lormeau

A modern JavasScript library for zipping and unzipping files

zip.js is a library for zipping and unzipping files. It supports UTF8 or ASCII filenames and offers a low-level API for writing and reading (big) zip files. This library is implemented with these HTML5 features: * Web workers * Typed arrays * Blob and BlobBuilder * FileReader * FileWriter

Install

include the zip.js file in your page
