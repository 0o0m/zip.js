zip.js

A JavaScript library to zip and unzip files

zip.js offers a low-level API for writing and reading large zip files (up to 4GB) with a stable RAM use. It also offers a Filesystem API in order to manipulate zip file structure.

This library is implemented with these HTML5 features:

  • Web workers
  • FileReader (and FileWriter optionally)
  • Blob and BlobBuilder
  • Typed arrays

Notes

  • This library works currently only with Chrome and Firefox (with DataView.js from David Flanagan).

Install

  • include zip.js, deflate.js, inflate.js and dataview.js (for Firefox DataView support) in your project
  • add zip.js in your HTML page 
    <script type="text/javascript" src="/lib/zip.js"></script>
  • add dataview.js if you need to support Firefox 
    <script type="text/javascript" src="/lib/dataview.js"></script>
  • set zip.workersScriptPath if needed (i.e. deflate.js and inflate.js path) 
    zip.workersScriptPath = "/lib/";
