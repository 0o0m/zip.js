zip.js

A JavaScript library to zip and unzip files

zip.js offers a low-level API for writing and reading large zip files (up to 4GB) with a stable RAM use. It also offers a Filesystem API in order to manipulate zip file structure.

This library is implemented with these HTML5 features:

  • Web workers
  • FileReader (and FileWriter optionally)
  • Blob and BlobBuilder
  • Typed arrays

Notes

  • This library works currently only with Chrome and Firefox (with DataView.js from David Flanagan).

Install

  • include zip.js, deflate.js, inflate.js and dataview.js (for Firefox DataView support) in your project
  • add zip.js in your HTML page 
    <script type="text/javascript" src="/lib/zip.js"></script>
  • add dataview.js if you need to support Firefox 
    <script type="text/javascript" src="/lib/dataview.js"></script>
  • set zip.workersScriptPath if needed (i.e. deflate.js and inflate.js path) 
    zip.workersScriptPath = "/lib/";

Core API

  • Reader and Writer constructors

    zip.js can handle multiple types of data thanks to a generic API. This feature is based on 2 abstract constructors: Reader and Writer.

    Reader constructors help to read data from a file and Writer constructors help to write data into a file. For example, if you need to read a zip in order to display it into a page, you must use a Reader to read the compressed zip data. You must also use a Writer to write uncompressed zip data.

  • Reader constructors

    zip.TextReader(text)

    string reader

    Parameters:

    text (string)
    The string object to read

    zip.BlobReader(blob)

    Blob object reader

    Parameters:

    blob (Blob)
    The Blob object to read

    zip.Data64URIReader(dataURI)

    Data URI reader

    Parameters:

    dataURI (String)
    The data URI to read

    zip.HttpReader(URL)

    HTTP content reader

    Parameters:

    URL (String)
    The URL of the content to read

    zip.HttpRangeReader(URL)

    HTTP content reader using "Range" request header

    Parameters:

    URL (String)
    The URL of the content to read

  • Writer constructors

    zip.TextWriter()

    String writer 


    zip.BlobWriter()

    Blob object writer 


    zip.FileWriter(fileEntry)

    File object writer

    Parameters:

    fileEntry (FileEntry)
    the FileEntry referencing the File object to write

    zip.Data64URIWriter(mimeString)

    Data URI writer

    Parameters:

    mimeString (String)
    the data MIME type - optional

  • Zip Reader creation

    zip.createReader(reader, callback, onerror)

    Create a ZipReader object.

    Parameters:

    reader (Reader)
    the Reader object used to read input data
    callback (Function)
    end callback function with new ZipReader object as parameter
    onerror (Function)
    the global error callback function - optional

  • Zip entries listing

    ZipReader.prototype.getEntries(callback)

    Get all entries from a zip.

    Parameters:

    callback (Function)
    callback function with an Array of Entry object as parameter

  • Zip entry data reading

    Entry

    Constructor representing a zip file entry.

    Attributes:

    filename (String)
    file name
    directory (boolean)
    true if the entry is a directory
    compressedSize (number)
    compressed data size
    uncompressedSize (number)
    uncompressed data size
    timeBlob (number)
    creation date in raw format
    crc32 (number
    data checksum

    Methods:

    getData(writer, onend, onprogress)

    Get the data from a file entry of the zip.

    Attributes:

    writer (Writer)
    Writer object used to write output data
    onend (Function)
    end callback function with output data as parameter
    onprogress (Function)
    progress callback function with an index (number) and a max value (number) as parameters - optional

  • Zip Reader closing

    ZipReader.prototype.close(callback)

    Close the opened zip.

    Parameters:

    callback (Function)
    end callback function

  • Zip Writer creation

    zip.createWriter(writer, callback, onerror)

    Create a ZipWriter object.

    Parameters:

    writer (Writer)
    the Writer object used to write output data
    callback (Function)
    end callback function with new ZipWriter object as parameter
    onerror (Function)
    the global error callback function - optional

  • Zip entry writing

    ZipWriter.prototype.add(name, reader, onend, onprogress, options)

    Add a new entry into the zip.

    Parameters:

    name (String)
    entry file name
    reader (Reader)
    the Reader object used to read entry data to add
    onend (Function)
    end callback function
    onprogress (Function)
    progress callback function with an index (number) and a max value (number) as parameters - optional
    options (Object)
    entry options
    directory (boolean)
    true if the entry is a directory
    level (number)
    compression level from 0 to 9

  • Zip Writer closing

    ZipWriter.prototype.close(callback)

    Close the opened zip.

    Parameters:

    callback (Function)
    end callback function

Examples

  1. Write a zip into a File object

    // use a FileWriter to store the zip into a FileEntry object (in file system)
zip.createWriter(new zip.FileWriter(fileEntry), function(writer) {

  // use a TextReader to read the String to add
  writer.add("filename.txt", new zip.TextReader("test!"), function() {
    // onend callback

    // close the zip writer
    writer.close(function(blob) {
      // blob contains the zip file as a Blob object

    });
  }, function(currentIndex, totalIndex) {
    // onprogress callback
  });
}, function(error) {
  // onerror callback
});

  2. Read a zip from a Blob/File object

    // use a BlobReader to read the zip from the Blob object
zip.createReader(new zip.BlobReader(blob), function(reader) {

  // get all entries from the zip
  reader.getEntries(function(entries) {
    if (entries.length) {

      // get entry content as text
      entries[0].getData(new zip.TextWriter(), function(text) {
        // text contains the entry data as a String

        // close the zip reader
        reader.close(function() {
          // onclose callback
        });

      }, function(current, total) {
        // onprogress callback
      });
    }
  });
}, function(error) {
  // onerror callback
});
